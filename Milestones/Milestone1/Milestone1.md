# Milestone 1: Data Collection and Understanding

## Objective
Collect and analyze smart home energy consumption data.

## Datasets
- HomeC.csv (Real dataset)
- energy_data.csv (Synthetic dataset)

## Data Processing
- Checked missing and duplicate values  
- Converted timestamps to datetime format  
- Set datetime index  
- Resampled data (hourly, daily, monthly)

## Exploratory Analysis
- Hourly, daily, and monthly consumption trends  
- Appliance-wise analysis  
- Distribution and variability analysis  
- Polynomial curve fitting  
- Rolling averages  
- Weekend vs Weekday analysis

## Mathematical & Statistical Methods
- Mean, median, standard deviation  
- Skewness and kurtosis  
- Correlation analysis  
- Polynomial regression

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- SciPy  
- Jupyter Notebook  
- VS Code

## Folder Structure
- data/ : Raw datasets  
- notebooks/ : Analysis notebooks  
- src/ : Python scripts  
- Milestone1.md : Documentation

## Outcome
Prepared a clean, structured, and well-analyzed dataset for future modeling and forecasting.  
Reviewed and organized folder structure.
