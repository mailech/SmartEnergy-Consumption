# Milestone 1: Data Collection and Understanding
---------------------------

 # Objective
-----------
To collect, preprocess, and analyze smart home energy consumption data to understand device-level usage patterns and prepare the dataset for machine learning modeling.
---------------------------------------------------------------------------
Datasets
--------------------
HomeC.csv (Real dataset).
Synthetic Energy Dataset (Generated dataset)
------------------------
Data Preprocessing
--------------------------
Checked for missing and duplicate values
Applied forward filling method (ffill()) to handle missing data
Converted timestamps to datetime format
Set datetime column as index
Resampled data for:
Hourly aggregation
Daily aggregation
Monthly aggregation
--------------------------------------

----------------------------

Time-series aggregation techniques
